# RDF Static Viz (React + Tailwind)
A React app that renders **static** SVG diagrams for small RDF subgraphs using D3 + ELK.js.

## Run
```bash
npm install
npm run dev
# open the printed local URL
```

## Use
- Click "RDF/JSON" to load either `public/examples/foaf.ttl` or `public/examples/graph.json`
- Optionally load `public/examples/config.json`
- Click **Render** and then **Download SVG**
