# Evaluation Pack

This folder contains the materials needed to write a complete Evaluation section:

- PROTOCOL.md: step-by-step evaluation protocol
- METRICS_DEFINITION.md: metric formulas and interpretations
- DATASETS.md: suggested datasets and sampling guidance
- COMPARISON_MATRIX.md: tool configuration matrix template
- RESULTS_TEMPLATE.csv: results table template for metrics
- REPORT_TEMPLATE.md: thesis-ready Evaluation section outline
- THREATS_TO_VALIDITY.md: threats and mitigations checklist

Suggested workflow:
1) Select datasets + sampling strategy (DATASETS.md).
2) Define evaluation protocol and runs (PROTOCOL.md).
3) Run this tool with Evaluation mode on and export metrics.
4) Collect results from other tools using the same inputs.
5) Fill COMPARISON_MATRIX.md and RESULTS_TEMPLATE.csv.
6) Draft the Evaluation section using REPORT_TEMPLATE.md.
